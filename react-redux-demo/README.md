
使用引入下面所有依赖
```
npm install
```

也可以单独一个个引入
```
// 用antd组件库
npm install antd --save
```
```
// 配置自定义
yarn add react-app-rewired customize-cra
```
```
// 组件按需加载
yarn add babel-plugin-import
```
```
// less 样式
yarn add less less-loader
```

```
// redux 相关
yarn add redux react-redux redux-devtools-extension --save
```
